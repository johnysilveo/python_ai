BULLS AND COWS GUI GAME

How to run:
1. Open this folder in PyCharm.
2. Run main.py.

Or from terminal:
python main.py

Files:
- main.py                         working Tkinter game
- assets/menu_screen.png          menu graphic
- assets/game_screen.png          game graphic
- assets/win_screen.png           win graphic
- assets/how_to_play_screen.png   instruction graphic
- assets/all_graphics_mockup.png  original full graphic sheet

Rule used:
Cows  = correct digit in the correct position.
Bulls = correct digit, but in the wrong position.
